# 2.shelljs

Shelljs 是 Node.js 下的脚本语言解析器，具有丰富且强大的底层操作(Windows/Linux/OS X)权限。Shelljs 本质就是基于 node 的一层命令封装插件，让前端开发者可以不依赖 linux 也不依赖类似于 cmder 的转换工具，而是直接在我们最熟悉不过的 javascript 代码中编写 shell 命令实现功能。
